# Dockerfile for PHP-FPM

PHP7.4-FPM base image with `mysqli`, `pdo_mysql`, `pgsql`, `pdo_pgsql`, `gd`, `zip`, `bz2`, `xsl`, `xDebug`, `bcmath`, `exif`, `intl`, `ImageMagick`, `imap`, `xmlrpc` and many others.
